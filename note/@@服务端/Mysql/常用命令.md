```mysql
# (1)、查看当前数据库中的表格（不会显示临时表）
SHOW TABLES;

# (2)、查看对应表的列 详情
SHOW COLUMNS FROM table_name;

# (3)、查看对应表格的 状态
SHOW TABLE STATUS LIKE 'table_name'\G;

# (4)、显示索引信息,可以通过添加 \G 来格式化输出信息。
SHOW INDEX FROM table_name; \G

# (5)、获取创建数据表(CREATE TABLE)的语句，该语句包含了原数据表的结构，索引等，\G 来格式化输出信息。
```
