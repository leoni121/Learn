### 1.1 Loaders ###

将 loaders 应用于最少数的必要模块中。而不是:

```js
{
  test: /\.js$/,
  loader: "babel-loader"
}
```

使用 `include` 字段仅将 loader 模块应用在实际需要用其转换的位置中：

```js
{
  test: /\.js$/,
  include: path.resolve(__dirname, "src"),
  exclude: /node_modules/,
  loader: "babel-loader"
}
```

### 1.2 Bootstrap ###

每个额外的 loader/plugin 都有启动时间。尽量少使用不同的工具。
