## Random.first()

* Random.first()

随机生成一个常见的英文名。

<!-- **使用示例**如下所示： -->

```js
Random.first()
// => "Nancy"
```

## Random.name( middle? )

* Random.name()
* Random.name( middle )

随机生成一个常见的英文姓名。

<!-- **参数的含义和默认值**如下所示： -->

### middle

可选。

布尔值。指示是否生成中间名。

<!-- **使用示例**如下所示： -->

```js
Random.name()
// => "Larry Wilson"
Random.name(true)
// => "Helen Carol Martinez"
```
