## 前端路由是直接找到与地址匹配的一个组件或对象并将其渲染出来。改变浏览器地址而不向服务器发出请求有两种方式:
   vue-router 提供了三种运行模式：
    hash: 使用 URL hash 值来作路由。默认模式。

    history: 依赖 HTML5 History API 和服务器配置。查看 HTML5 History 模式。

    abstract: 支持所有 JavaScript 运行环境，如 Node.js 服务器端。
