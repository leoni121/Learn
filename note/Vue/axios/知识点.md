## 1. axios 概述 ##

一个基于Promise 的可以在浏览器和nodejs中使用的HTTP 库

**axios 特点**

* 从浏览器中创建 XMLHttpRequests
* 从 node.js 创建 http 请求
* 支持 Promise API
* **拦截请求和响应** （就是有interceptor）
* **转换请求数据和响应数据**
* 取消请求
* 自动转换 JSON 数据
* 客户端支持防御 XSRF
