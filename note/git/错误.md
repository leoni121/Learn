### 2.  git pull报错:There is no tracking information for the current branch ###

是因为本地分支和远程分支没有建立联系 (使用git branch -vv 可以查看本地分支和远程分支的关联关系) .根据命令行提示只需要执行以下命令即可

```cmd
git branch --set-upstream-to=origin/远程分支的名字  本地分支的名字
```
